# Interpretation Limits

This workflow is designed for research use and careful outbreak analysis. It is not a clinical diagnostic system or an official public-health reporting platform.

## Segment-Specific Interpretation

Orthohantaviruses have S, M, and L genome segments. These segments can carry different phylogenetic signals. Do not infer transmission chains from one segment without checking whether the other available segments support the same relationship.

## Reassortment

Segment reassortment can create incongruent S, M, and L trees. Treat segment discordance as an analytic signal that needs review, not as a nuisance to hide by concatenating segments.

## Context Tree

The pan-Orthohantavirus context S tree is for placement, naming stability, and QC. It is not the primary outbreak interpretation surface.

## Time-Scaled Trees

Time-scaled analyses should use stricter inclusion criteria than exploratory ML trees. Records with ambiguous dates, uncertain sampling locations, or weak temporal signal should not drive public claims about timing or transmission.
